Real-Estate-Market-Transaction-Analysis-using-SQL

Project Overview
This project analyzes real estate transactions using SQL to uncover pricing trends, buyer behavior, housing supply, affordability, and investment patterns.

Objectives
- Explore real estate transaction data
- Clean and validate the dataset
- Analyze market trends
- Identify buyer behavior
- Study housing supply
- Evaluate affordability
- Generate business insights

The project follows a complete data analytics workflow:

- Data Exploration
- Data Cleaning
- Market Trend Analysis
- Buyer Behavior
- Housing Supply Analysis
- Buyer Segmentation
- Business Insights

Skills Demonstrated

- SQL Joins
- CTEs
- Window Functions
- Aggregate Functions
- CASE Statements
- Data Cleaning
- Ranking Functions
- Date Functions
- Business Analysis

Database
MySQL


Key Insights

- Highest priced cities
- Fastest growing markets
- Buyer segmentation
- Affordability analysis
- Investor activity
- Housing supply trends
